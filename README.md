## Repository Structure

| Folder / File | Description |
|---|---|
| `dataset/EMT_data.csv` | Raw data as obtained from the experimentalist |
| `dataset/EMT.csv` | Cleaned dataset used for modeling |
| `External Validation/` | Code and data for external validation on literature-reported syntheses |
| `Filtered_dataset/` | Code and data for the filtered dataset analysis |
| `Model Benchmark/` | Python package and scripts for model benchmarking |
| `Prediction for Experimentally Verified Synthesis/` | Code and data for predicting experimentally verified synthesis conditions |
