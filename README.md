## Repository Structure

| Folder / File | Description |
|---|---|
| `Dataset/EMT_data.csv` | Raw data as obtained from the experimentalist |
| `Dataset/EMT.csv` | Cleaned dataset used for modeling |
| `External_Validation/` | Code and data for external validation on literature-reported syntheses |
| `Filtered_dataset/` | Code and data for the filtered dataset analysis |
| `Model_Benchmark/` | Python package and scripts for model benchmarking |
| `Experimentally_Verified_Synthesis/` | Code and data for predicting experimentally verified synthesis conditions |
