## Repository Structure

| File/Folder | Description |
|---|---|
| `EMT_data.csv` | Raw data as obtained from the experimentalist |
| `EMT.csv` | Cleaned dataset used for modeling |
| `External Validation.zip` | Code and files for external validation on literature-reported syntheses |
| `Filtered_dataset.zip` | Code and files for the filtered dataset analysis |
| `Model Benchmark.zip` | Code and files for model benchmarking |
| `Prediction for Experimentally Verified Synthesis.zip` | Code and files for predicting experimentally verified synthesis conditions |
